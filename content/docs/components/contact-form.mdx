A complete contact form component built with React Hook Form and Zod validation. Features real-time validation, error handling, and clean UI design.

<ComponentPreview component='example-form' />

## Features

- **Form Validation**: Built-in validation using Zod schema
- **Error Handling**: Real-time validation with clear error messages
- **Accessibility**: Proper labeling and ARIA attributes
- **TypeScript**: Fully typed with TypeScript
- **Responsive**: Works on all screen sizes

## Usage

This component demonstrates best practices for form handling in React applications, including:

- Form state management with React Hook Form
- Schema validation with Zod
- Error message display
- Form submission handling
- Accessible form design

## Dependencies

- `zod` - Schema validation
- `@hookform/resolvers/zod` - React Hook Form Zod resolver
- `react-hook-form` - Form state management
